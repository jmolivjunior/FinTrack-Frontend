# FinTrack Frontend

Personal finance management app built with React and TypeScript.

## Technologies

- React 18
- TypeScript
- Tailwind CSS
- Axios
- React Router DOM

## Features

- Login and authentication
- Dashboard with transactions
- Add and delete transactions
- Monthly summary (income, expenses, balance)
- Protected routes with JWT

## Getting Started

### Prerequisites
- Node.js v22+
- FinTrack API running on http://localhost:5003
